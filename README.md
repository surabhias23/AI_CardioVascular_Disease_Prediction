# AI_CardioVascular_Disease_Prediction_Surabhi

This project performs data preprocessing, visualization, correlation analysis, and machine learning classification using a cardiovascular dataset.
The system predicts the risk of heart disease based on patient health parameters.

# Output

Data visualizations

Correlation matrix

Accuracy comparison of machine learning models

Heart disease prediction based on user input

# This project uses supervised learning:

Logistic Regression

K-Nearest Neighbors (KNN)

Support Vector Machine (SVM)

Decision Tree

Random Forest
